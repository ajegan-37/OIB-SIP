# 🐉 Dragon Booster Landing Page

A responsive and animated landing page inspired by the Dragon Booster theme.
This project is built using **HTML and CSS**, focusing on layout design, styling, and basic animations.

---

## 📌 Project Overview

This landing page showcases a fictional Dragon Booster universe with sections like hero banner, dragon types, and abilities.
The main goal of this project is to practice **UI design, layout structuring, and CSS animations**.

---

## 🚀 Features

* Responsive layout using Flexbox
* Smooth hover animations on cards and buttons
* Animated background effects
* Structured sections (Hero, Dragons, Powers, Battle)
* Clean and modern UI design

---

## 🛠 Technologies Used

* HTML5
* CSS3

---

```
project-folder/
│── index.html

```
